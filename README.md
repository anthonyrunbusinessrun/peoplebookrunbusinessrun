# PeopleBook — Ray Land Inc.
BOSS Recruiting & Talent Pipeline Portal

## Stack
- Next.js 14 App Router
- PostgreSQL + Prisma ORM
- Tailwind CSS
- NextAuth.js
- Airtable API (appGGFKuFxQ3Z0Wuz)
- Railway (90747dc5-257d-4af7-a29c-c77ae5e20cd9)

## Branding
Midnight Blue #0a1628 | Crimson Red #c0152a | Gold #c9a84c

## Getting Started
```bash
npm install
cp .env.example .env.local
# Fill in your env vars
npx prisma generate && npx prisma db push
npm run dev
```
